# Moony Budget Buddy

A personal budgeting app that helps you track expenses, set budgets, and build healthy financial habits.

## Features

- Track daily expenses by category
- Set monthly and yearly budgets
- Visual spending insights
- Rewards and streaks for staying on budget
- All data stored locally on your device — no account needed

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Running locally

```sh
npm install
npm run dev
```
